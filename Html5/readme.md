## Konfiguration
### View in Browser
```json
 "extension.viewInBrowser":"chrome",
```
### Debug in Chrome
Chrome muss wie folgt gestartet werden
```
"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
```
